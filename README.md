## 🚀 Instrucciones de Ejecución

Puedes ejecutar este proyecto utilizando un entorno de desarrollo integrado (IDE) o directamente desde la línea de comandos. Elige el método que prefieras:

### Opción 1: Ejecutar desde NetBeans

Dado que el proyecto utiliza **Maven**, NetBeans lo reconocerá automáticamente sin necesidad de configuraciones adicionales.

1. **Abre** NetBeans.
2. Ve al menú superior y selecciona **File** (Archivo) > **Open Project** (Abrir Proyecto).
3. **Selecciona la carpeta raíz** del proyecto (aquella donde se encuentra el archivo `pom.xml`).
4. En el panel lateral de **Projects** (Proyectos), despliega la estructura del proyecto.
5. Ve a **Source Packages** (Paquetes de fuentes) y expande el paquete **`Programa`**.
6. Haz **clic derecho** sobre el archivo `Programa.java`.
7. Selecciona la opción **Run File** (Ejecutar Archivo) o utiliza el atajo de teclado `Mayús + F6`.
