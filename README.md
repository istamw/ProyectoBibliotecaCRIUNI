=========================================================
SISTEMA DE GESTIÓN DE BIBLIOTECA
=========================================================

https://github.com/user-attachments/assets/27c6d42a-21f2-4c39-acce-223ee77a4398

Este es un proyecto Java estructurado con Maven que simula el manejo de un sistema de biblioteca (alumnos, libros, préstamos, etc.) mediante una interfaz de línea de comandos (CLI).

REQUISITOS PREVIOS:
- Java JDK 8 o superior instalado.
- Pantalla (opcional)

La clase principal (Main) que arranca el programa se encuentra en:
Paquete: Programa
Clase: Programa.java
(Ruta exacta: ```src/main/java/Programa/Programa.java```)

---------------------------------------------------------
OPCIÓN 1: EJECUTAR DESDE NETBEANS
---------------------------------------------------------
Como el proyecto está hecho con Maven, NetBeans lo reconocerá automáticamente.

1. Abre NetBeans.
2. Ve a "File" (Archivo) > "Open Project" (Abrir Proyecto).
3. Selecciona la carpeta raíz del proyecto (donde se encuentra el archivo pom.xml).
4. En el panel de "Projects" (Proyectos), despliega la carpeta del proyecto.
5. Ve a "Source Packages" (Paquetes de fuentes) > paquete "Programa".
6. Haz clic derecho sobre el archivo "Programa.java".
7. Selecciona la opción "Run File" (Ejecutar Archivo) o presiona Mayús + F6.

---------------------------------------------------------
OPCIÓN 2: EJECUTAR DESDE LA TERMINAL
---------------------------------------------------------
1. Abre tu terminal.
2. Navega hasta la carpeta "src/main/java" dentro del proyecto:
   ```cd ruta/al/proyecto/src/main/java```

3. Compila todos los archivos Java (esto creará los archivos .class):
   ```javac Programa/Programa.java```

4. Ejecuta el programa indicando el nombre del paquete y la clase:
   ```java Programa.Programa```
